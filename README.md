# 预测帝

**一句话：把「谁赢」这个问题，变成一份有数据、有逻辑、有概率的专业预测报告。**

预测帝是专为 AI Agent 打造的事件预测 Skill。你只管问，它自动搜赔率、查 Elo、翻战绩、读新闻，多源融合后给你一份像分析师写出的结论——**零配置，开箱即用。**

---

## 核心能力

### 多源情报，一网打尽

不拍脑袋，不凭感觉。预测帝并行拉取 **五路信号**，交叉验证：

| 信号 | 能力 |
|------|------|
| 博彩赔率 | 读懂市场真金白银的共识 |
| 预测市场 | 捕捉 Polymarket 等事件合约价格 |
| Elo 评分 | 用实力模型量化强弱差距 |
| 历史统计 | 近绩、交锋、主客场一手掌握 |
| 新闻情绪 | 伤病、变阵、舆论风向实时纳入 |

缺一路？自动降级，绝不硬编。

### 确定性融合，拒绝 LLM 瞎猜

融合概率由 **`fuse_signals.py` 脚本** 算出，不是模型随口一说。赔率归一化、Elo 换算、加权融合——全程可复现、可审计。

### 四步流水线，问完就出报告

```
你的问题 → 事件识别 → 数据采集 → 预测引擎 → 专业报告
```

从「湖人能赢吗」到「今天世界杯四场怎么猜」，同一套流程，稳定输出。

### 结构化报告，像分析师写的

- 结论 + 融合概率 + 置信度
- 多源对比表（一眼看出分歧）
- 关键依据 & 风险点
- 完整数据溯源 + 免责声明

## 怎么用

**对用户：** 直接问，或调用 Skill：

```
/yucedi 预测今天世界杯比赛
/yucedi 湖人能赢凯尔特人吗？
```

**对 Agent：** 加载 `SKILL.md`，按 Checklist 执行即可。

**环境要求：** Python 3，无第三方依赖，无需 API Key。

## 目录一览

```
yucedi/
├── README.md          ← 你正在看的
├── SKILL.md           ← Agent 执行手册
├── onboarding.md      ← 首次使用：环境自检 + 引导配置 Polymarket（agent 剧本）
├── examples.md        ← 完整示例
├── references/        ← 数据源指南 & 报告模板
├── relay/             ← 国内用户用的 Polymarket 中转模板（Cloudflare Worker / VPS）
└── scripts/           ← 赔率 / Elo / Polymarket / 融合 等脚本 + setup_polymarket.py（自检配置）
```
